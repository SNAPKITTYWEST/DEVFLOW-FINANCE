# Contributing to Collective Kitty

## TypeScript Excellence Commitment
We maintain a zero-tolerance policy for type safety violations. 
Every contribution must utilize Branded Types for domain identifiers to prevent cross-contamination. 
Switch-case blocks must be exhaustive using the `never` type pattern. 
Interfaces are preferred for public API surface definitions to ensure documentation clarity. 
TypeScript is our law, ensuring the sovereign ledger remains immutable and reliable.
